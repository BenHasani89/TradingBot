# Tradingbot

Automatisierter Swing-Trading-Bot fuer Kryptowaehrungen und Aktien. Arbeitet ausschliesslich mit
simuliertem Geld (Paper-Trading), bis eine vollstaendige Pruefung (siehe Projekt-Roadmap) und eine
ausdrueckliche Freigabe vorliegen.

## Status

Projekt befindet sich in der Planungs-/Aufbauphase. Der aktuelle Entwicklungsstand ist in
[docs/phasen/](docs/phasen/) dokumentiert (eine Datei pro Phase, fortlaufend nummeriert).

## Dokumentation

- [docs/phasen/](docs/phasen/) - Projektentwicklung, eine Datei pro Phase (Architektur, Technologie,
  Umgebung, ...)
- [docs/handbuch/technisch/](docs/handbuch/technisch/) - technisches Handbuch fuer Entwickler
- [docs/handbuch/benutzer/](docs/handbuch/benutzer/) - Benutzerhandbuch (folgt erst ab Phase 9)

## Einrichtung

Siehe [docs/phasen/03_entwicklungsumgebung.md](docs/phasen/03_entwicklungsumgebung.md) fuer die

## Wichtiger Hinweis

Dieses System handelt ausschliesslich mit Spielgeld, bis in der Projekt-Roadmap ausdruecklich anders
festgelegt. Es stellt keine Anlageberatung dar.
