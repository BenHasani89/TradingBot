# Benutzerhandbuch (Kunden/Mitarbeiter)

Status: Platzhalter, angelegt in Phase 3. **Noch ohne Inhalt.**

## Wichtiger Hinweis, bevor dieses Handbuch befuellt wird

Dieses Handbuch ist erst relevant, sobald das System tatsaechlich von Kunden oder Mitarbeitern
(nicht nur vom Projektinhaber selbst) genutzt wird. Bevor das so weit ist, muessen zwei Dinge
geklaert sein - siehe die kritische Pruefung in
[docs/phasen/03_entwicklungsumgebung.md](../../phasen/03_entwicklungsumgebung.md), Abschnitt 1:

1. **Rechtliche Pruefung** (Schweiz): Klaerung, ob und welche Bewilligungs- oder
   Meldepflichten (z. B. nach FINIG/FIDLEG) und welche datenschutzrechtlichen Pflichten
   (revDSG) bei einer Nutzung durch Dritte entstehen - durch eine Fachperson, nicht durch
   dieses Projekt.
2. **Technische Voraussetzungen**: Benutzerverwaltung, Zugriffskontrolle und Mandantenfaehigkeit
   existieren im aktuellen Entwicklungsstand noch nicht (bewusste Entscheidung, siehe Phase 3).

Der Inhalt dieses Handbuchs (Dashboard-Bedienung in einfacher Sprache, ohne technische Details)
wird ab Phase 9 (Dashboard) entworfen, aber erst nach positivem Ergebnis von Punkt 1 tatsaechlich
fuer eine Nutzung durch Dritte freigegeben.
